
# PiscaLuzesLed

## 📋 Descrição
Projeto de IoT desenvolvido em aula, implementando um sistema de controle de LEDs com efeito de pisca-pisca.

## 🛠️ Tecnologias Utilizadas
- **Linguagem**: C++
- **Plataforma de Simulação**: Tinkercad
- **Tipo de Projeto**: IoT (Internet of Things)

## 📅 Data
Aula de IoT - Hoje

## 🔌 Componentes
- Arduino/Microcontrolador
- LEDs
- Resistores
- Fios de conexão

## 💻 Como Usar
1. Acesse o projeto no Tinkercad
2. Compile o código em C++
3. Execute a simulação
4. Observe o padrão de pisca-pisca dos LEDs
